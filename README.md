# Zenow Front End

Template cloned from https://github.com/erikras/react-redux-universal-hot-example

## Installation

```bash
npm i
```

## Dev
```bash
npm run dev
```

## Production
```bash
npm run build
npm start
```

## Evironment Variables
```
API_URL=http://localhost:3030
CLIENT_SECRET=...
ENV=dev
NODE_ENV=development
SESSION_SECRET=...
```
